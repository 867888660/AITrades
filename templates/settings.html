Configuration Center

Settings

保存后会直接驱动后端实时采集器。

LLM Provider

内置调查 Agent 和 EventGraph Proposal 会使用这里的 OpenAI-compatible 模型配置；第一版仅启用阿里百炼兼容模式。

API Key 会和现有行情密钥一样保存到本地加密 secrets 文件；EventGraph 正式入图仍必须经过 Proposal/人工确认。

Agent 权限边界

这些设置会直接限制 `/api/agent/*` 能力；批准、实盘执行和策略注册仍只能由人类完成。

EventGraph 审批模式

只管理 EventGraph 正式入图；新闻 observation / derived event 继续自动入库，策略审批和实盘执行不受这里影响。

完全信任会让通过校验的 EventGraph change request 由 system 自动 approve/apply，并写入版本历史和审计日志。

资金与风险上限

Agent 超过这些边界时，风控会阻止提交或要求人工修改。

扫描与提案默认值

Agent 做热门市场扫描和批量草案时，会优先使用这些默认参数。

能力开关

关闭后，对应 agent 接口会返回权限错误；人类审批和人工参数修改不受影响。

硬性禁止：agent 不能批准自己的策略、不能直接执行实盘、不能修改自身权限。
